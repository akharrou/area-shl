## USECASES

----
#### To


```bash
#ℹ︎ get/print/display external internet protocol address (IP)
❯ dig +short myip.opendns.com @resolver1.opendns.com
```


This only works if your ISP doesn’t replace DNS requests (which it shouldn’t).
