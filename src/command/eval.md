# `eval`

## BRIEF

    ❯ eval [EXPR]

Read `EXPR`s as input to the shell and execute the resulting command(s).

## INSTALLATION


```bash
#ℹ︎ installation
❯ *comes builtin*
```


## USECASES

----
#### To evaluate shellscript expressions:


```bash
#ℹ︎ evaluate shell script commands/arguments expressions
❯ eval "echo hi"
```
