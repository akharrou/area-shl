# `mdimport`

## BRIEF

`mdimport` is used to test Spotlight plug-ins, list the installed
plug-ins and schema, and re-index files handled by a plug-in when
a new plug-in is installed.

## INSTALLATION


```bash
#ℹ︎ installation
❯ *comes pre-installed*
```
