# `source`

## BRIEF

Read and execute shellscript files.

## INSTALLATION


```bash
#ℹ︎ installation
❯ *comes builtin*
```


## USECASES

----
#### To "source", i.e read and execute commands from file:


```bash
#ℹ︎ read and execute commands from files
❯ source <FILE>
```


```bash
#ℹ︎ read and execute executable files
❯ ./EXECUTABLE
```


```bash
#ℹ︎ read and execute commands from files
❯ . <FILE>
```



```bash
#ℹ︎ ?
❯ eval <FILE>
```
