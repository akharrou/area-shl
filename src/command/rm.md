# `rm`

## INSTALLATION


```bash
#ℹ︎ installation of `rm`
❯ *comes pre-installed*
```


## USECASES

----
#### To delete/remove/erase files and directories:


```bash
#ℹ︎ delete/remove/erase file
❯ rm FILE
```


```bash
#ℹ︎ delete/remove/erase directory
❯ rm -r DIR
```
