# `zsh/computil`

## BRIEF

The `zsh/computil` module defines low level functions used by completion functions in the completion system:

- `comparguments`
- `compdescribe`
- `compfiles`
- `compgroups`
- `compquote [-p] NAMES ...`
- `comptags`
- `comptry`
- `compvalues`
