# `kill`

## BRIEF

Utility that sends signals to processed specified by `pid`.

## INSTALLATION


```bash
#ℹ︎ installation
❯ *comes builtin*
```


## USECASES

----
#### To kill/terminate processed by specifying its `pid` (process identifier):


```bash
#ℹ︎ terminate process specified by <pid> (process identifier)
❯ kill -9 <pid>
```

    kill(1)
