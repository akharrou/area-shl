# `hostname`

## INSTALLATION


```bash
#ℹ︎ installation of `hostname`
❯ *comes pre-installed*
```


## USECASES

----
#### To get/print current system hostname:


```bash
#ℹ︎ get/print current system hostname
❯ hostname -s
```

    Mac


```bash
#ℹ︎ get/print current system hostname, with domain
❯ hostname -f
```

    Mac.local


```bash
#ℹ︎ set current system hostname
❯ sudo hostname NAME
```
