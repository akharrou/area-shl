# `vim` (`vi` Improved)

## INSTALLATION


```bash
#ℹ︎ installation of `vim`
❯ brew install vim
```


## USECASES

----
#### To edit files:


```bash
#ℹ︎ edit file
❯ vim <FILE>
```



# ALIASES

    alias vim="vim -u $HOME/.config/vim/vimrc.vim"
