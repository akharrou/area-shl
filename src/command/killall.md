# `killall`

## BRIEF

Kill/terminates all processes that match given name.

```bash
#ℹ︎ installation

```


## USECASES

----
#### To kill all processes with specified name:


```bash
#ℹ︎ terminate all processes specified by process name
❯ killall <process-name>
```
