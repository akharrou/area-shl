## USECASES

----
#### To get/print/display last N (specified) file lines:


```bash
#ℹ︎ get/print/display last N (specified) file lines
❯ tail -3 <FILE>
```


```bash
#ℹ︎ get/print/display watch <FILE> data in real time refresh
❯ tail -f <FILE>
```
