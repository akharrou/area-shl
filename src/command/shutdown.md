# `shutdown`

## INSTALLATION


```bash
#ℹ︎ installation of `shutdown`
❯ *comes pre-installed*
```


## USECASES

----
#### To shutdown/halt/stop OR reboot/restart computer:


```bash
#ℹ︎ halt/stop computer now
❯ shutdown -h now
```


```bash
#ℹ︎ reboot/restart computer now
❯ shutdown -r now
```
