```bash
#ℹ︎ installation of `cat`

```


## USECASES

----
#### To print files:


```bash
#ℹ︎ print/display files
❯ cat FILE
```


```bash
#ℹ︎ print/display files, with numbered lines
❯ cat -n FILE
```


```bash
#ℹ︎ print/display files, with numbered non-empty lines
❯ cat -b FILE
```
