# `rmdir`

## INSTALLATION


```bash
#ℹ︎ installation of `rmdir`
❯ *comes pre-installed*
```


## USECASES

----
#### To remove/delete empty directories:


```bash
#ℹ︎ remove/delete empty directory
❯ rmdir DIR
```


```bash
#ℹ︎ remove/delete empty directories path
❯ rmdir -p PATH/DIR
```
