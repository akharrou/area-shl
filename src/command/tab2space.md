# `tab2space`

## INSTALLATION


```bash
#ℹ︎ installation of `tab2space`
❯ *comes pre-installed*
```


## USECASES

----
#### To expand tabs and set line endings of text files:


```bash
#ℹ︎ convert/expand text file tabs to spaces
❯ tab2space -t2 <FILE>
```


```bash
#ℹ︎ convert/set line endings to specified type [unix|mac|dos]
❯ tab2space -unix <FILE>
```
