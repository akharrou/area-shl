# Literal bar (`|`)

## BRIEF

`|` is a control operator.

## INSTALLATION


```bash
#ℹ︎ installation of literal bar (`|`)
❯ *comes builtin*
```


## USECASES

----
#### To form a command pipeline, i.e a series of commands that each passes its output to the next as input:

    ❯ echo hi | cat
    hi
