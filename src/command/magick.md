# `magick`

## INSTALLATION


```bash
#ℹ︎ installation of `imagemagick`
❯ brew install magick
```


## USECASES

----
#### To transform digital images:


```bash
#ℹ︎ compress image to specified percentage
❯ magick mogrify -resize 60% IMAGE
```
